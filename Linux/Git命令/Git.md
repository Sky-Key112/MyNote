# GIT

[**`在线Git可视化教程`**](https://learngitbranching.js.org/)

## 命令

    git add     #添加文件
    git commit  #提交
    git push    #推送到远程
    git pull    #拉取远程
    git clone   #下载
    git status  #查看状态
    git log     #查看日志
    git diff    #查看差异
    git rm      #删除文件
    git mv      #移动文件
    git stash   #提交到暂存区
    git stash list #查看暂存区
    git reset   #恢复暂存区
    git checkout #切换到暂存区
    git merge   #合并
    git branch  #列出分支
    git branch -d #删除分支
    git branch -a #列出所有分支
